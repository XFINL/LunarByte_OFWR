# LunarByte_Zero 官方網站產品需求文檔

## 1. 產品概述
LunarByte_Zero 是一款創新的音樂軟體官方網站，展示軟體功能、開發團隊、下載連結及開源詳情。網站採用極致絲滑的滾動動畫效果，配合高玻璃態設計美學，營造神秘、高端的品牌形象。目標用戶為音樂愛好者和技術開發者。

## 2. 核心功能

### 2.1 用戶角色
| 角色 | 需求描述 | 核心權限 |
|------|----------|----------|
| 訪客 | 瀏覽網站資訊、查看下載連結 | 僅瀏覽 |
| 開發者 | 了解開源詳情、查看技術棧 | 瀏覽 + 下載 |

### 2.2 功能模組
1. **首頁**：單頁式應用，包含所有內容區塊
2. **導航選單**：懸浮式頂部導航，含語言切換
3. **內容展示**：滾動觸發的動畫效果

### 2.3 頁面詳情
| 區塊名稱 | 模組描述 | 功能特性 |
|----------|----------|----------|
| Hero區塊 | Logo + 副標題 | 漸入動畫、視差效果 |
| 軟體介紹 | 功能特性展示 | 滾動淡入、卡片佈局 |
| 開發團隊 | 團隊成員卡片 | 網格排列、懸停動畫 |
| 下載區 | 平台下載按鈕 | 玻璃按鈕、狀態標識 |
| 開源詳情 | 技術棧說明 | 程式碼片段、連結 |

## 3. 核心流程
用戶訪問網站 → 瀏覽內容 → 切換語言 → 查看下載連結 → 了解開源詳情

## 4. 用戶介面設計

### 4.1 設計風格
- **主色調**：純黑 `#000000`、純白 `#FFFFFF`
- **玻璃效果**：高透明度 `backdrop-filter: blur(20px)`、高圓角 `border-radius: 24px`
- **字體**：現代無襯線字體（使用Google Fonts）
- **圖標**：阿里巴巴圖標庫（iconfont）
- **裝飾**：無emoji，使用純圖形元素

### 4.2 動畫設計
- **滾動觸發**：元素進入視口時淡入上移
- **懸停效果**：玻璃卡片輕微放大、陰影加深
- **過渡曲線**：`cubic-bezier(0.4, 0, 0.2, 1)` 追求絲滑感
- **動畫時長**：300-500ms 為宜

### 4.3 響應式設計
- **桌面優先**：1200px 以上完整展示
- **平板適配**：768px - 1199px 單列佈局
- **手機優化**：320px - 767px 堆疊式佈局
