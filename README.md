# LunarByte_Zero

Modern music app website built with React 18 + TypeScript

## Features

- Multi-language support (Traditional Chinese, English, Japanese, Russian)
- Glass morphism UI design
- Smooth scroll animations
- Responsive design

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Zustand (State Management)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

The website is optimized for deployment on Vercel, Netlify, or any static hosting service.

## GitHub Push Instructions

Since the GitHub CLI requires authentication, you can push manually:

```bash
# Create a new repository on GitHub first, then:

git remote add origin https://github.com/YOUR_USERNAME/lunarbyte-zero.git
git branch -M main
git push -u origin main
```

Or use GitHub CLI with authentication:

```bash
# Authenticate with GitHub
gh auth login

# Create and push to repository
gh repo create lunarbyte-zero --public --source=. --push
```
